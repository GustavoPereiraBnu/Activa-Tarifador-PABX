# Activa Tarifador + Activa Coletor

Sistema de tarifação e coleta de bilhetes para **PABX Leucotron iSion**.

## Download

Baixe a versão mais recente do instalador:

[Setup-ACTIVA-Tarifador.exe](https://github.com/GustavoPereiraBnu/Activa-Tarifador-PABX/releases/latest/download/Setup-ACTIVA-Tarifador.exe)

## O que o sistema faz

- coleta os bilhetes de chamadas do PABX;
- importa os registros para banco Firebird;
- gera relatórios por ramal;
- exibe resumo de chamadas por destino;
- realiza rateio mensal da conta telefônica;
- mostra painel de chamadas simultâneas.

## Instalação

1. Baixe o instalador.
2. Execute como administrador.
3. Escolha os módulos que deseja instalar ou atualizar.
4. Conclua a instalação.

## Módulos

- **Activa Coletor**: responsável pela coleta dos bilhetes do PABX.
- **Activa Tarifador**: responsável pela importação, armazenamento e emissão de relatórios.

## Requisitos

- Windows 64 bits
- Firebird configurado no ambiente
- PABX Leucotron iSion
